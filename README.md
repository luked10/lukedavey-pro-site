# lukedavey-pro-site

Professional portfolio site for Luke Davey.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion

## Structure

- app/layout.tsx — global shell and navigation
- app/page.tsx — homepage content
- app/route-shell.tsx — route transitions
- app/globals.css — global styles and typography
- tailwind.config.ts — design tokens and content paths
